# Mittal Acquisition - Project Conventions

## Project Context

This project contains M&A due diligence documents and transaction materials for The Kidney Experts, PLLC acquisition of Mittal Kidney & Dialysis PLLC.

## Document Standards

### Format
- All documents in Markdown format
- Use tables for structured data
- Include source citations where applicable
- Date all materials

### Naming Convention
- Documents: `NN-document-name.md` (e.g., `01-executive-summary.md`)
- Templates: `template-name.md`
- Research: `source-notes.md`

### Required Sections
Each document should include:
1. Title and version
2. Last updated date
3. Purpose/objective
4. Main content
5. Action items (if applicable)
6. Sources/references
