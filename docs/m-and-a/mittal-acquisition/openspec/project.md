# Mittal Kidney & Dialysis Acquisition Project

## Project Overview

**Project Name**: Mittal Kidney & Dialysis Acquisition
**Type**: M&A Due Diligence & Transaction Execution
**Status**: Pre-NDA / Discovery Phase
**Created**: January 2026

## Transaction Summary

| Element | Details |
|---------|---------|
| **Buyer** | The Kidney Experts, PLLC (TKE) |
| **Target** | Mittal Kidney & Dialysis PLLC |
| **Seller's Rep** | Jud Cannon, CPA - Canon Wright Blount, Memphis, TN |
| **Asking Price** | $10,000,000 - $12,500,000 |
| **EBITDA Multiple** | 6-8x |
| **Implied EBITDA** | ~$1,560,000 - $1,670,000 |
| **Recommended Target** | $8,500,000 - $10,500,000 (5.5-6.5x) |

## Strategic Rationale

1. **Memphis Market Entry**: Access to major metro with high CKD/ESRD burden
2. **Mississippi Expansion**: First state outside Tennessee via Southaven location
3. **Provider Scale**: Combined 5 MDs + 7 APPs
4. **Revenue Acceleration**: $12-15M combined revenue potential
5. **DaVita Relationships**: Medical directorship revenue + dialysis partnerships
6. **RPM Infrastructure**: Operational NSight program to scale

## Target Practice Profile

- **Founded**: 2004 by Dr. Sanjeev Mittal, MD
- **Locations**: 2 (Cordova, TN + Southaven, MS)
- **Providers**: 3 MDs + 2 NPs
- **EHR**: eClinicalWorks
- **Hospital Systems**: Baptist Memorial, Methodist Le Bonheur, St. Francis

## Key Success Factors

1. DaVita medical directorship contract transferability
2. Provider retention (especially Dr. Mittal transition plan)
3. EHR migration from eClinicalWorks to CKD EHR by DaVita
4. SBA 7(a) financing approval
5. Seller note structure (20%+)

## Critical Risks

| Risk | Severity | Mitigation |
|------|----------|------------|
| DaVita contracts non-assignable | HIGH | Condition close on consent; earnout structure |
| Provider concentration (Mittal) | HIGH | Verify revenue by provider; retention agreements |
| EHR integration complexity | MEDIUM | Budget $200-500K; plan 20% productivity dip |
| Southaven MS startup losses | MEDIUM | Normalize in EBITDA; model 12-month ramp |
| Valuation premium (6-8x) | MEDIUM | Counter with 5.5-6.5x; justify with earnout |
