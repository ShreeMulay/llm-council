## Slide 8: Three Trust Destroyers

```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│    THREE PATHS THAT DESTROY TRUST                              │
│                                                                │
│    ❌ IGNORING                                                 │
│       Hoping the problem goes away.                            │
│       Creates resentment. Nothing changes.                     │
│                                                                │
│    ❌ GOSSIPING                                                │
│       Talking ABOUT someone instead of TO them.                │
│       Even if they agree, you both know you can't be trusted.  │
│                                                                │
│    ❌ ESCALATING                                               │
│       Asking the boss to handle it for you.                    │
│       Signals you won't work things out directly.              │
│                                                                │
│    All three lower short-term anxiety,                         │
│    create long-term damage.                                    │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

**Speaker Notes**: When there's an issue, these are the three easy paths. They all feel safer in the moment. They all destroy trust over time. We've all done them. Starting today, we stop.

## Slide 10: Two Rules for Direct Conversation

```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│    RULE 1: SPEAK FOR YOURSELF                                  │
│                                                                │
│    ❌ "We've noticed you've been missing deadlines."           │
│    ✅ "I noticed you missed the deadline. What happened?"      │
│                                                                │
│    No "we think" or "everyone feels."                          │
│    Own YOUR perspective with "I" statements.                   │
│                                                                │
│    ─────────────────────────────────────────────               │
│                                                                │
│    RULE 2: STATEMENTS BEFORE QUESTIONS                         │
│                                                                │
│    ❌ "Why didn't you follow the protocol?"                    │
│    ✅ "I saw you handled that differently. I'm curious         │
│        about your reasoning."                                  │
│                                                                │
│    Share the thought that led to your question FIRST.          │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

**Speaker Notes**: These two rules prevent most communication breakdowns. "We think" makes people defensive. Leading with questions makes intent ambiguous. Fix both.

## Facilitator Guide

### Before the Session
- [ ] Print wallet cards for all attendees
- [ ] Have posters ready to display
- [ ] Prepare culture document copies for reference
- [ ] Set up room for discussion (circle or U-shape preferred)

### During the Session
- **Slides 1-6**: Set context (10 min)
- **Slides 7-10**: Accountability deep-dive (15 min) — THIS IS CRITICAL
- **Slides 11-18**: Values and Fundamentals (15 min)
- **Slides 19-22**: Application (10 min)
