## Design Specifications

### Dimensions
- **Size**: 24" x 36" (standard poster size)
- **Orientation**: Portrait
- **Margins**: 1" all sides
- **Safe area**: Content within 22" x 34"

### Colors

| Element | Color | Hex Code |
|---------|-------|----------|
| Background | White | #FFFFFF |
| Primary Text | Dark Navy | #1E3A5F |
| Headers | Deep Teal | #0D9488 |
| Accent Lines | Light Gray | #E5E7EB |
| Trust Destroyers | Muted Red | #DC2626 |
| Trust Builders | Deep Green | #059669 |
| Box Backgrounds | Light Cream | #FEFCE8 |

### Typography

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Company Name | Sans-serif (Inter/Helvetica) | 48pt | Bold |
| Slogan | Sans-serif | 24pt | Bold |
| Taglines | Sans-serif | 18pt | Medium |
| Section Headers | Sans-serif | 28pt | Bold |
| Body Text | Sans-serif | 14pt | Regular |
| Quote | Serif (Georgia/Merriweather) | 16pt | Italic |
| Footer | Sans-serif | 12pt | Light |

### Visual Elements
- Thin horizontal lines (1pt) between sections
- Rounded corners on boxes (8px radius)
- Subtle drop shadow on main content boxes
- Company logo in top right corner (if available)
- QR code in bottom corner linking to full culture doc (optional)

---

## Alternative Sizes

### 18" x 24" (Smaller spaces)
- Scale all fonts down 25%
- Same layout, tighter spacing

### 11" x 17" (Tabloid/Ledger)
- For printing on standard office equipment
- Scale fonts down 50%
- Consider splitting into 2 posters if readability suffers

### 8.5" x 11" (Letter size handout)
- Use simplified single-column layout
- Prioritize 6 Fundamentals + 5 Questions + Contract
- Remove values section (covered in full doc)

---

## Print Instructions

1. **Paper**: Matte or satin finish poster paper (100+ GSM)
2. **Lamination**: Optional matte lamination for durability in high-traffic areas
3. **Mounting**: Foam board mounting recommended for professional appearance
4. **Quantity**: 
   - Each clinic location: 2-3 posters (break room, conference room, hallway)
   - Each department: 1 poster
   - Administrative offices: 1 poster

---

## Digital Display Version

For TVs or digital signage in waiting rooms:
- Create 1920x1080 (16:9) version
- Rotate between sections every 15-30 seconds
- Add subtle animations (fade transitions)
- Consider patient-friendly version (just values + tagline, not internal accountability details)
