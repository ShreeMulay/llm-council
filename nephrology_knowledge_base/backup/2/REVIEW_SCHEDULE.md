## Review Process Workflow

### 1. Pre-Review Preparation (2 weeks before review date)
- [ ] Check for new clinical guidelines (KDIGO, KDOQI, ASN, NKF)
- [ ] Review FDA drug approvals relevant to nephrology
- [ ] Search PubMed for landmark trials published since last review
- [ ] Check CMS/Medicare policy updates (for practice management docs)

### 2. Document Review (Review month)
- [ ] Read entire document for accuracy and currency
- [ ] Update treatment protocols with new evidence
- [ ] Revise dosing recommendations if changed
- [ ] Add new diagnostic criteria or classifications
- [ ] Update cross-references to other documents
- [ ] Verify all external links still functional
- [ ] Check for outdated terminology or deprecated practices

### 3. Post-Review Actions
- [ ] Update "Last Updated" date in YAML frontmatter
- [ ] Set "Next Review" date based on review cycle
- [ ] Document changes in CHANGELOG.md
- [ ] Update version number if major changes made
- [ ] Notify stakeholders of significant clinical practice changes

### 4. Emergency Updates (Outside Regular Cycle)
Trigger immediate review if:
- FDA boxed warning or drug withdrawal
- Major guideline revision (e.g., KDIGO CKD 2024)
- Landmark trial results change standard of care (e.g., DAPA-CKD)
- CMS policy change affecting reimbursement
- Safety alert or public health emergency
