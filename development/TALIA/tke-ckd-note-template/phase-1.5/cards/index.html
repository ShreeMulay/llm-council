Print Instructions

  1. Click "Print All Cards" below or use Ctrl+P / Cmd+P
  2. Set paper size to Letter (8.5" x 11")
  3. Enable "Background graphics" in print settings for colors
  4. Each card prints at 5.5" x 8.5" (half-letter size)
  5. Cut along the center line to separate cards
  6. For individual cards, click the card link below
