## Tier 1 Screening (No Separate Billing)

Tier 1 screening (grip strength, sit-to-stand, gait speed) is performed by MAs as part of the vital signs workflow. It is included in the E/M visit and does NOT generate a separate charge.

## Tier 2 Assessment - CPT Codes

### Physical Performance Testing

| CPT Code | Description | Work RVU | Facility Rate | Non-Facility Rate |
|----------|-------------|----------|---------------|-------------------|
| 97750 | Physical performance test or measurement with written report, each 15 minutes | 0.45 | ~$18 | ~$25 |

**Usage**: Bill 97750 for each 15-minute increment of testing. A full Tier 2 assessment (SPPB + 6MWT + TUG) typically takes 30-45 minutes = 2-3 units.

**Documentation requirements**:
- Written report of findings
- Interpretation of results
- Recommendations

### Physical Therapy Evaluation

| CPT Code | Description | Work RVU | Facility Rate | Non-Facility Rate |
|----------|-------------|----------|---------------|-------------------|
| 97161 | PT evaluation, low complexity | 1.20 | ~$48 | ~$85 |
| 97162 | PT evaluation, moderate complexity | 1.50 | ~$60 | ~$105 |
| 97163 | PT evaluation, high complexity | 1.80 | ~$72 | ~$125 |

**Complexity Determination**:

| Complexity | History | Examination | Clinical Decision Making |
|------------|---------|-------------|-------------------------|
| Low (97161) | 1-3 elements | 1-2 body systems | Low complexity |
| Moderate (97162) | 4+ elements | 3+ body systems | Moderate complexity |
| High (97163) | 4+ elements | 4+ body systems | High complexity |

**Typical Tier 2 Assessment**: 97162 (moderate complexity) is most appropriate for standard referrals.

## Documentation Requirements

### For 97750 (Physical Performance Test)

**Required elements**:
1. Indication for testing
2. Tests performed with specific protocols used
3. Quantitative results (times, distances, scores)
4. Interpretation of results
5. Comparison to normative data or prior results
6. Recommendations based on findings

**Sample documentation**:
```
Physical Performance Testing (97750 x 2, 30 minutes)

Indication: Tier 1 screening abnormal (grip strength 22 kg, below threshold)

Tests performed:
- Short Physical Performance Battery (SPPB): 7/12
  - Balance: 3/4 (tandem 6 sec)
  - Gait speed: 2/4 (6.5 sec = 0.62 m/s)
  - Chair stand: 2/4 (14.2 sec)
- 6-Minute Walk Test: 320 meters
- Timed Up and Go: 14.2 seconds

Interpretation: Intermediate physical performance with moderate functional 
limitation. Results consistent with probable sarcopenia in setting of CKD G4.
6MWT distance below 350m threshold associated with increased CV mortality.
TUG >12 sec indicates increased fall risk.

Recommendations:
1. Physical therapy referral for strengthening program
2. Nutrition consult for protein optimization
3. Repeat assessment in 3 months
```

### For PT Evaluation (97161-97163)

**Required elements**:
1. History (onset, prior treatment, comorbidities)
2. Systems review
3. Tests and measures performed
4. Evaluation/diagnosis
5. Prognosis
6. Plan of care with goals

### Medical Necessity Statement

For insurance authorization, include:

```
Medical Necessity: Patient with CKD stage [X] demonstrates objective 
evidence of functional decline with [grip strength/gait speed/SPPB] 
below established thresholds. Physical performance assessment is 
medically necessary to:
1. Quantify degree of functional impairment
2. Identify fall risk
3. Guide rehabilitation interventions
4. Monitor response to treatment

Without intervention, patient is at increased risk for falls, 
hospitalization, and accelerated functional decline.
```

## Billing Tips

### Maximize Reimbursement

1. **Document time accurately** - 97750 is time-based (per 15 min)
2. **Use appropriate complexity** - Don't under-code PT evaluations
3. **Include all diagnoses** - Primary + secondary codes
4. **Write detailed reports** - Supports medical necessity
5. **Track outcomes** - Improvement supports continued treatment

### Common Denials and Solutions

| Denial Reason | Solution |
|---------------|----------|
| Not medically necessary | Add medical necessity statement, cite evidence |
| Duplicate service | Ensure different dates or different tests |
| Missing documentation | Include all required elements |
| Incorrect modifier | Use appropriate modifiers (59, 76, etc.) |
| Exceeded frequency | Document why additional testing needed |

### Modifier Usage

| Modifier | Description | Use When |
|----------|-------------|----------|
| 59 | Distinct procedural service | Multiple tests same day |
| 76 | Repeat procedure by same physician | Follow-up testing |
| 77 | Repeat procedure by different physician | Different provider |
| GP | Services delivered under PT plan | PT-directed services |

## Payer-Specific Notes

### Medicare

- 97750 covered when medically necessary
- PT evaluation covered with appropriate diagnosis
- DXA for sarcopenia may require LCD review
- CCM codes require patient consent

### Commercial Payers

- Coverage varies by plan
- Prior authorization may be required for PT
- Document medical necessity thoroughly
- Appeal denials with clinical evidence

### Medicare Advantage

- Follow plan-specific guidelines
- May have different prior auth requirements
- Network restrictions may apply
