# MEDICATION DOSING IN CKD MODULE

## Card Overview

| Attribute | Value |
|-----------|-------|
| **Type** | Module Card |
| **Color** | Green (#27AE60) |
| **Size** | 5" × 7" |
| **Paper** | 80-100 lb cardstock, matte |
| **Used By** | MA at Station 2 / Pharmacist Review |
| **Priority** | 2 |
| **Phase** | Phase 2 (Full Pilot) |

## Activation Trigger

- CKD Stage 3b or higher (GFR < 45)
- On dialysis
- New medication added to regimen
- Medication reconciliation visit
- Adverse drug reaction suspected
- Hospitalization with new medications

## Purpose

Review and optimize medication dosing for kidney function, identify nephrotoxic medications that should be avoided or dose-adjusted, and prevent adverse drug events. Many medications require dose adjustment in CKD to prevent toxicity.

## Training Notes

### For MAs
1. **NSAIDs are the #1 problem** - always ask about pain medication use
2. **OTC and supplements count** - many patients don't mention these
3. **Metformin cutoffs**: Max 1000 mg/day for GFR 30-45, STOP if <30
4. **Gabapentin needs big dose reductions** - common in our population
5. **Document everything** - changes need to go to PCP and pharmacy
