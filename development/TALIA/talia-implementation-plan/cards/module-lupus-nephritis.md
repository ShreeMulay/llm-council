# LUPUS NEPHRITIS MODULE

## Card Overview

| Attribute | Value |
|-----------|-------|
| **Type** | Module Card |
| **Color** | Purple-Pink (#E91E63) |
| **Size** | 5" × 7" |
| **Paper** | 80-100 lb cardstock, matte |
| **Used By** | MA at Station 2 |
| **Priority** | 2 |
| **Phase** | Phase 2 (Full Pilot) |

## Activation Trigger

- Diagnosis of systemic lupus erythematosus (SLE) with kidney involvement
- Lupus nephritis (any class)
- History of kidney biopsy showing lupus nephritis
- On immunosuppression for lupus nephritis
- Anti-dsDNA positive or low complement with proteinuria

## Purpose

Specialized monitoring for lupus nephritis patients requiring coordination between nephrology and rheumatology, immunosuppression management, disease activity monitoring, and prevention of kidney failure. Lupus nephritis affects 50% of SLE patients.

## Training Notes

### For MAs
1. **Know the class** - determines treatment intensity
2. **Track serologies** - C3, C4, anti-dsDNA indicate activity
3. **UPCR is key** - remission means <500 mg/g
4. **Plaquenil eye exams** - essential safety monitoring
5. **Coordinate with rheumatology** - this is a shared patient

### Lupus Nephritis Class Quick Reference
| Class | Severity | Treatment |
|-------|----------|-----------|
| I, II | Mild | May only need supportive care |
| III, IV | Severe | Requires immunosuppression |
| V | Variable | Often needs immunosuppression |
| VI | End-stage | Limited options, chronic damage |

### Signs of Flare vs Remission
| Flare | Remission |
|-------|-----------|
| Rising UPCR | UPCR <500 mg/g |
| Active sediment | Inactive sediment |
| Rising anti-dsDNA | Stable or negative anti-dsDNA |
| Falling C3/C4 | Normal C3/C4 |
| Declining GFR | Stable GFR |

### Red Flags
- UPCR doubling in short period
- New RBC casts (active nephritis)
- Rapidly declining GFR
- Fever on immunosuppression (infection)
