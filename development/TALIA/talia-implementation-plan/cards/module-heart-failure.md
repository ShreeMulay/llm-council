# HEART FAILURE MODULE

## Card Overview

| Attribute | Value |
|-----------|-------|
| **Type** | Module Card |
| **Color** | Red (#E74C3C) |
| **Size** | 5" × 7" |
| **Paper** | 80-100 lb cardstock, matte |
| **Used By** | MA at Station 2, Provider at Station 4 |
| **Priority** | 5 |
| **Phase** | Phase 2 (Full Pilot) |

## Activation Trigger

- Heart failure diagnosis (HFrEF, HFmrEF, or HFpEF)
- Elevated BNP/NT-proBNP
- On heart failure medications (diuretics for HF, carvedilol, entresto)
- Cardiomegaly on imaging

## Purpose

Manage cardiorenal syndrome through coordinated heart and kidney care. Optimize GDMT, manage volume status, and prevent hospitalizations.

## Training Notes

**For MAs:**
1. Weight is CRITICAL - always compare to dry weight goal
2. Ask about pillows needed to sleep (orthopnea) 
3. Ask about walking distance - quantify symptoms
4. Check legs for edema (correlate with Measurement Card)
5. Note any recent hospitalizations for HF

**Volume Assessment Clues:**
- Hypervolemic: weight up, edema increased, SOB worse, JVP elevated
- Euvolemic: at dry weight, no symptoms at rest
- Hypovolemic: weight below dry weight, dizzy, low BP, leg cramps

**GDMT Targets (for provider):**
- All HFrEF patients should be on 4-pillar therapy: BB + ACEi/ARB/ARNI + MRA + SGLT2i
- Uptitrate to target doses when tolerated
- SGLT2i now indicated for HF regardless of diabetes status

**Success Metrics:**
- 4-pillar GDMT prescription rate
- Hospital readmission rate
- Weight stability at dry weight
