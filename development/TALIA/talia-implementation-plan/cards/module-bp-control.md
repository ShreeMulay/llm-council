# BP CONTROL MODULE

## Card Overview

| Attribute | Value |
|-----------|-------|
| **Type** | Module Card |
| **Color** | Dark Blue (#2C3E50) |
| **Size** | 5" × 7" |
| **Paper** | 80-100 lb cardstock, matte |
| **Used By** | MA at Station 2 |
| **Priority** | 4 |
| **Phase** | Phase 2 (Full Pilot) |

## Activation Trigger

- BP >140/90 at any visit
- BP >130/80 with diabetes or CKD (tighter goal)
- On 3+ blood pressure medications
- History of uncontrolled hypertension

## Purpose

Achieve and maintain blood pressure control to slow CKD progression. Track home BP readings, optimize medication regimen, address adherence barriers.

## Training Notes

**For MAs:**
1. Always ask about home BP monitoring
2. Request BP log if patient monitors at home
3. List ALL BP medications - count them
4. Ask about adherence without judgment
5. Note if patient needs BP cuff recommendation

**Home BP Technique Check:**
- Correct cuff size (bladder encircles 80% of arm)
- Arm at heart level
- Seated quietly for 5 minutes before
- No caffeine 30 min before
- Empty bladder first

**BP Goals by Condition:**
- CKD with proteinuria: <130/80
- CKD without proteinuria: <130/80 to <140/90
- Diabetes: <130/80
- Elderly (>80): may accept higher targets

**Success Metrics:**
- Percentage of patients at BP goal
- Home monitoring adoption rate
- Adherence improvement
