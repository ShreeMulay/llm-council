# PRE-DIALYSIS MODULE

## Card Overview

| Attribute | Value |
|-----------|-------|
| **Type** | Module Card |
| **Color** | Navy (#34495E) |
| **Size** | 5" × 7" |
| **Paper** | 80-100 lb cardstock, matte |
| **Used By** | MA at Station 2, Provider at Station 4 |
| **Priority** | 7 |
| **Phase** | Phase 3 (Iterate) |

## Activation Trigger

- CKD Stage 4-5 (GFR <30)
- eGFR <20
- Anticipated dialysis within 12 months
- AV fistula or graft in place
- PD catheter in place

## Purpose

Prepare patients for optimal dialysis start. Ensure timely access planning, modality education, and comprehensive pre-dialysis preparation to avoid emergent dialysis starts.

## Training Notes

**For MAs:**
1. Track GFR trend over time - calculate rate of decline
2. Ensure modality education has been done
3. Check access status at every visit
4. For patients with fistula - check thrill and document
5. Verify Hep B vaccination status

**Critical Timelines:**
- GFR <25: Discuss transplant referral
- GFR <20: Initiate access planning (fistula takes 3-6 months)
- GFR <15: Access should be in place, modality decided
- GFR <10-12: Ready to start dialysis if symptomatic

**Fistula Check:**
- Feel for thrill (vibration) at fistula site
- Note any swelling, redness, or tenderness
- Ask about arm pain or weakness
- Remind patient to protect fistula arm

**Success Metrics:**
- Planned dialysis start rate (vs emergency)
- Fistula first rate
- Hep B vaccination completion
- Modality education completion
