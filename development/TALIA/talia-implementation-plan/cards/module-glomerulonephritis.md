# GLOMERULONEPHRITIS MODULE

## Card Overview

| Attribute | Value |
|-----------|-------|
| **Type** | Module Card |
| **Color** | Purple (#8E44AD) |
| **Size** | 5" × 7" |
| **Paper** | 80-100 lb cardstock, matte |
| **Used By** | MA at Station 2 |
| **Priority** | 2 |
| **Phase** | Phase 2 (Full Pilot) |

## Activation Trigger

- Diagnosis of glomerulonephritis (any type)
- Nephrotic syndrome or nephritic syndrome
- Significant proteinuria (>500 mg/day)
- Hematuria with proteinuria
- On immunosuppressive therapy for kidney disease
- History of kidney biopsy showing glomerular disease
- IgA nephropathy, FSGS, membranous nephropathy, etc.

## Purpose

Monitor and manage patients with glomerular diseases which can lead to rapid kidney function decline. Track proteinuria, immunosuppression side effects, and disease activity to prevent progression to kidney failure.

## Training Notes

### For MAs
1. **UPCR is the key marker** - always get the most recent value
2. **Know the syndrome type** - helps classify severity
3. **Immunosuppression = infection risk** - always ask about fevers/illness
4. **Drug levels matter** - tacrolimus, cyclosporine need monitoring
5. **Proteinuria goal is <500 mg/g** - or 50% reduction from baseline

### Remission Definitions
| Status | Criteria |
|--------|----------|
| Complete remission | UPCR <300 mg/g + stable GFR |
| Partial remission | UPCR <3500 mg/g AND >50% reduction |
| Active disease | Not meeting above criteria |

### Red Flags - Notify Provider
- Rapidly rising creatinine (AKI on GN)
- Gross hematuria (visible blood)
- UPCR >10,000 mg/g (severe nephrotic)
- Fever in immunosuppressed patient
- New or worsening edema despite therapy
