# TRANSPLANT EVALUATION MODULE

## Card Overview

| Attribute | Value |
|-----------|-------|
| **Type** | Module Card |
| **Color** | Teal (#1ABC9C) |
| **Size** | 5" × 7" |
| **Paper** | 80-100 lb cardstock, matte |
| **Used By** | MA at Station 2, Provider at Station 4 |
| **Priority** | 8 |
| **Phase** | Phase 3 (Iterate) |

## Activation Trigger

- CKD Stage 4+ (GFR <30)
- GFR <25
- Existing transplant referral
- Patient inquiring about transplant
- On transplant waiting list

## Purpose

Facilitate timely transplant referral and track progress through the transplant evaluation process. Identify barriers and support patients through listing.

## Training Notes

**For MAs:**
1. Track evaluation progress at every visit
2. Ask about any completed tests since last visit
3. Inquire about living donor possibilities each visit
4. Note any barriers and escalate to social work
5. Keep transplant center contact information updated

**Referral Timing:**
- Refer when GFR <25 (or earlier if rapidly declining)
- Preemptive transplant (before dialysis) is ideal
- Living donor evaluation can proceed at higher GFR

**Key Centers for Jackson, TN:**
- Vanderbilt University Medical Center (Nashville)
- Methodist University Hospital (Memphis)
- Baptist Memorial Hospital (Memphis)
- Consider multi-listing for shorter wait

**Success Metrics:**
- Referral rate at GFR <25
- Evaluation completion rate
- Time from referral to listing
- Living donor identification rate
