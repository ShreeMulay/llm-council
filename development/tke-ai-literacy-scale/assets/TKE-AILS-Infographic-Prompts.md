## Usage Instructions

### For Nano Banana / Gemini Image Generation:

1. **Copy the Global Style Context** from above
2. **Paste it at the start of your prompt**
3. **Add the specific infographic prompt** below it
4. **Generate and iterate** - ask for adjustments:
   - "Make the text larger"
   - "Use more contrast"
   - "Simplify the layout"
   - "Emphasize the purple color more"

### Tips for Best Results:

- **Be patient** - first generation rarely perfect
- **Iterate 3-5 times** minimum
- **Save good versions** before requesting changes
- **Specify dimensions** if the tool supports it
- **Request PNG format** for quality

### Post-Generation Checklist:

- [ ] Text is readable at intended print/display size
- [ ] Colors match TKE brand palette
- [ ] Logo/branding is correctly placed
- [ ] No AI hallucinations in text content
- [ ] Professional healthcare aesthetic maintained
- [ ] Footer includes TKE attribution
