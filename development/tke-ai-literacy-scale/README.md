## Overview

TKE-AILS (The Kidney Experts AI Literacy Scale) is a comprehensive framework for measuring, developing, and advancing AI collaboration skills across all roles at The Kidney Experts, PLLC.

This initiative aligns with our core value: **"AI Amplifies Human Impact"** — technology handles routine work so humans can do remarkable work.

## The 6 Levels of AI Literacy

| Level | Name | Description |
|-------|------|-------------|
| 0 | **Unaware** | No AI exposure; may fear job replacement |
| 1 | **Aware** | Knows AI exists; uses basic tools (voice assistants) |
| 2 | **Explorer** | Experiments with AI; writes basic prompts |
| 3 | **Practitioner** | Uses AI regularly; crafts effective prompts; verifies outputs |
| 4 | **Integrator** | Embeds AI into workflows; teaches others |
| 5 | **Champion** | Drives adoption; develops custom workflows; shapes strategy |

---

## The 3 Dimensions

1. **AI Mindset** — Treating AI as teammate, not just tool (10× better outcomes)
2. **AI Capability** — Practical skills: prompting, verification, iteration
3. **AI Application** — Role-specific use cases in nephrology context

---

## The 5 Prompting Rules

1. **Be Specific** — Include context, details, and desired outcome
2. **Provide Context** — Explain the clinical scenario and audience
3. **TKE Workspace Only** — PHI is safe HERE (BAA with Google), never on personal accounts
4. **Verify Everything** — AI is a teammate, not the final signer
5. **Iterate & Refine** — Chat with it to improve the output
