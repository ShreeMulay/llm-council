# Nephrology ICD-10 and CPT Code Quick Reference

This comprehensive coding reference is designed for nephrologists, Advanced Practice Providers (APPs), and clinical staff at The Kidney Experts (TKE). Accurate coding ensures proper reimbursement, reflects patient complexity (HCC risk adjustment), and supports compliance with CMS guidelines.

***

## 1. CKD Staging & AKI Codes (ICD-10)

Accurate CKD staging requires documentation of the eGFR threshold. **Code first** any underlying condition (e.g., diabetes, hypertension).

| Stage / Condition | ICD-10 Code | Clinical Definition / eGFR Threshold |
| :--- | :--- | :--- |
| **CKD Stage 1** | N18.1 | Kidney damage with normal eGFR (≥90 mL/min/1.73m²) |
| **CKD Stage 2** | N18.2 | Kidney damage with mild ↓ eGFR (60-89 mL/min/1.73m²) |
| **CKD Stage 3a** | N18.31 | Mild to moderate ↓ eGFR (45-59 mL/min/1.73m²) |
| **CKD Stage 3b** | N18.32 | Moderate to severe ↓ eGFR (30-44 mL/min/1.73m²) |
| **CKD Stage 3 (Unsp)**| N18.30 | Unspecified stage 3 (Avoid if possible; specify 3a/3b) |
| **CKD Stage 4** | N18.4 | Severe ↓ eGFR (15-29 mL/min/1.73m²) |
| **CKD Stage 5** | N18.5 | eGFR <15 mL/min/1.73m² (Not on dialysis) |
| **ESRD** | N18.6 | End-Stage Renal Disease (Requires chronic dialysis) |
| **CKD Unspecified** | N18.9 | Chronic kidney disease, unspecified |
| **AKI** | N17.9 | Acute kidney injury, unspecified |
| **ATN** | N17.0 | Acute kidney failure with tubular necrosis |

*Note: For patients on chronic dialysis, always use N18.6, not N18.5, and add Z99.2 (Dependence on renal dialysis).*

## 3. Diabetic Kidney Disease (DKD)

**Coding Rule:** CMS requires combination codes for Diabetes and CKD. You must assume a causal relationship unless the provider specifically documents that the CKD is *not* related to the diabetes. **Always use an additional code to identify the CKD stage (N18.x).**

| Condition | ICD-10 Code | Required Additional Code |
| :--- | :--- | :--- |
| **T2DM with CKD** | E11.22 | Add N18.x for CKD stage |
| **T2DM with Nephropathy** | E11.21 | Add N18.x for CKD stage |
| **T2DM with Hyperglycemia**| E11.65 | Use if poorly controlled |
| **T1DM with CKD** | E10.22 | Add N18.x for CKD stage |
| **T1DM with Nephropathy** | E10.21 | Add N18.x for CKD stage |

## 9. Common CPT Codes for Nephrology

### Evaluation & Management (E&M)
*Based on Medical Decision Making (MDM) or Total Time.*

*   **New Patient Outpatient:** 99202, 99203, 99204, 99205
*   **Established Patient Outpatient:** 99212, 99213, 99214, 99215
*   **Initial Inpatient Care:** 99221, 99222, 99223
*   **Subsequent Inpatient Care:** 99231, 99232, 99233
*   **Inpatient Consultations:** 99252, 99253, 99254, 99255 *(Note: Medicare and many commercial payers no longer accept consult codes; use Initial Inpatient codes instead).*

### Dialysis Services (MCP - Monthly Capitation Payment)
*   **Outpatient ESRD (Age 20+):**
    *   90960 (4+ visits per month)
    *   90961 (2-3 visits per month)
    *   90962 (1 visit per month)
*   **Inpatient Hemodialysis:**
    *   90935 (Single evaluation by physician during HD)
    *   90937 (Repeated evaluations by physician during HD)

### Procedures & Infusions
*   **Venipuncture:** 36415 (Routine), 36416 (Capillary)
*   **Renal Biopsy:** 50200 (Percutaneous, by trocar or needle)
*   **IV Infusion (Initial up to 1 hr):** 96365
*   **IV Infusion (Each add'l hr):** 96366

### Common Nephrology HCPCS (Drug) Codes
*Always document exact dosage administered and wastage to support billing.*
*   **Venofer (iron sucrose):** J1756 *(Typical dose: 100-200 mg IV)*
*   **Injectafer (ferric carboxymaltose):** J1439 *(Typical dose: 750 mg IV)*
*   **Rituxan (rituximab):** J9312 *(Typical dose: 1000 mg IV or 375 mg/m2)*
*   **Krystexxa (pegloticase):** J2507 *(Typical dose: 8 mg IV every 2 weeks)*

## 11. Documentation Tips & Compliance Flags

### The MEAT Criteria for HCC Coding
To ensure chronic conditions (like CKD, Diabetes, HF) are captured for risk adjustment, documentation must show that the condition was addressed during the encounter using the **MEAT** criteria:
*   **M**onitor: Ordering labs (e.g., BMP, UACR), reviewing eGFR trends.
*   **E**valuate: Assessing symptoms, physical exam findings (e.g., edema).
*   **A**ssess: Documenting status (e.g., "CKD stage 3b, stable," "DKD progressing").
*   **T**reat: Prescribing/adjusting meds (e.g., starting Farxiga (dapagliflozin), adjusting lisinopril), dietary counseling.

### ⚠️ Compliance & Audit Flags
*   **"History of" vs. "Active":** Do not document "History of CKD" if the patient currently has CKD. "History of" implies the condition is resolved.
*   **Linkage:** Clearly link conditions. Write "CKD secondary to Type 2 Diabetes" rather than listing them as separate, unrelated bullet points.
*   **Copy/Paste Cloning:** Avoid pulling forward outdated lab values or physical exam findings in the EMR. Always update the eGFR and UACR to the most recent values.

### Patient Education Talking Points (Billing)
*   **"Why did I get a separate bill for the infusion?"** -> *"Your visit today included an evaluation by the provider (E&M) and the actual administration of the medication (e.g., Venofer (iron sucrose)). Insurance requires us to bill these as two separate services."*
*   **"What is this charge for a 'monthly capitation'?"** -> *"For our dialysis patients, insurance bundles our care into a monthly charge based on how many times we round on you at the dialysis center, rather than billing you for every single conversation."*
